Project_Three: Hindi Alphabet Test

HindiAlphabetStudy.zip

This was a recreational project that I had made during my first semester of learning Java. During this time, I had an interest in learning the Hindi language, and the first step of this process was learning the alphabet. Although this project is very elementary in its execution as is evident by the use of English pronunciation in place of actual Hindi characters and its heavy reliance on user input, I thought this project was notable because it was the first instance in which I found myself able to use what I learned in a CS class to develop myself in other areas of interest. Furthermore, after reviewing this projects some time later, I realized that even though this was written during my first year of coding, this project does do somethings well. For example, while testing you from a list of inputs from a local file, it chooses the inputs randomly and without repititon; additionally, after you finish testing yourself on the letters, it informs you about which letter you struggled the most with and asks the user if they want a graph depicting which words they got wrong. Using all the knowledge I had, I constructed an alphabet test which, in my opinion, aged well for what it was.
